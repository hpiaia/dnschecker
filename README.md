# Usage
```sh-session
$ npm install -g dnschecker
$ dnschecker COMMAND
running command...
$ dnschecker (-v|--version|version)
dnschecker/0.0.1 darwin-x64 node-v15.6.0
$ dnschecker --help [COMMAND]
USAGE
  $ dnschecker COMMAND
...
```
